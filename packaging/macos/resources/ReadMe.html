BQST 1.0.0

BQST combines a broad Baxandall-style EQ with two saturation modes for mix bus, master bus, and group processing.

Installed Files

  • VST3: /Library/Audio/Plug-Ins/VST3/BQST.vst3
  • Audio Unit: /Library/Audio/Plug-Ins/Components/BQST.component

Presets

Factory presets are included inside the plugin. User presets are saved as .bqstpreset files in the user's application data folder.

Requirements

This release is built for macOS on Apple Silicon.
