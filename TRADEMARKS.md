# BQST Branding And Trademark Guidelines

BQST is a project by Rohan Kulshrestha.

The BQST name, logo, visual identity, plugin UI artwork, installer artwork, website images,
factory presets, screenshots, and related branding assets are reserved. You may not use them
to present modified versions, derivative plugins, commercial products, or unrelated projects
as official BQST releases.

You may refer to the project as BQST when discussing, reviewing, linking to, or contributing
to the original project, provided the reference is truthful and does not imply endorsement.

Forks and modified builds should use a clearly different name and visual identity.
