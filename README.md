# BQT

BQT is a planned JUCE/C++ audio plugin: a dual Baxandall-style mastering EQ with independent saturation per side.

The plugin is intended to support L/R and M/S operation, realtime/render oversampling up to 8x, optional saturation autogain, and a two-slot 500-series-style UI.

See [docs/project-plan.md](docs/project-plan.md) for the current product and implementation plan.
